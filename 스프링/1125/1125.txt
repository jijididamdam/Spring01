회원관리 프로그램 작성
회원은 아이디, 이름, 전화번호, 포인트(기본값 100), 등록일(시스템 날짜 자동등록), select로 Vip와 일반선택이 가능하도록

create table vipcus(
id varchar2(10) primary key,
name varchar2(10),
tel varchar2(13),
point number default 100,
dt date default sysdate,
grade char(4) check(grade in('vip','일반'))
);



1. 메인화면
메인화면은 그림하나 넣기, 헤더, nav, footer구분되게 하시오.

2. 메뉴는 홈으로 게시판 회원가입 회원목록
3. 회원가입한다.
vip일 경우는 포인트를 1000으로 하고, 일반일 경우는 기본값이 저장되게 한다.

4. 회원가입 후에는 메인화면으로 돌아 온다.

5. 회원전체보기는
아이디, 이름, 전화번호, 포인트 수정/삭제
  abc   홍길동  010        1000        m/d

6. 이름을 입력하면 새창으로 회원의 정보를 볼 수 있고, 닫기 버튼을 만들어서 창을 닫을 수 있다.

7. m을 클릭하면 수정하겠습니까? 라는 자바스크립트 확인 창이 나오고 비밀번호를 입력 후 비밀번호가
맞으면 수정화면이 나오고 수정이 가능하도록 하시오

8. d를 클릭하면 삭제하겠습니까? 라는 자바스크립트 확인 창이 나오고 비밀번호를 입력후 삭제가 가능하도록 하시오
